# 🎓 EduPlatform – Full Stack Learning & Internship System

EduPlatform is a **full-stack web application** inspired by platforms like Coursera, designed to provide an interactive learning experience for **students, teachers, and admins**.

It supports course creation, enrollment, certifications, internships, chat system, and much more — all in one platform.

---

## 🚀 Features

### 👨‍🎓 Student Features

* Register/Login
* Explore & enroll in courses
* Personalized **dashboard**
* View enrolled courses & progress
* Download **certificates**
* Give Instructor * Course **Feedback** 
* Apply for **internships**
* Get Certificate For **Course** & **Internship**
* Chat with teachers (WhatsApp-style system)
* Chat with admin (Support-style system)
* Search & connect with users
* Profile management (edit, password, email)
* Course recommendations

---

### 👨‍🏫 Teacher Features

* Register/Login with **OTP VERIFICATION**
* Create & publish courses
* Upload course content & materials
* **Edit** course content & materials
* Manage enrolled students
* Upload notes & resources
* Interact with students via chat
* View Feedback
* Profile management

---

### 🛠️ Admin Features

* Approve & manage courses
* Manage users (students & teachers)
* Control internships
* Design & manage **certificates**
* Monitor platform activity
* Dashboard with analytics
